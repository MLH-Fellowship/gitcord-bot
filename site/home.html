GitCord GitHub Devpost

GitCord Bot

The ultimate Discord Bot that provides you with a number of developer tools and helps you manage your GitHub activity from Discord.

Integrations

GitCord has a few integrations that allow you to retrieve content from external platforms such as Github, Wappalyzer and Lighthouse.

Utilities

GitCord has utilities to help you post quickly on GitHub. They include posting comments on issues and pull requests.

Website Commands

You can run the following commands to get lighthouse stats and tech stack for websites:
-lighthouse (get lighthouse stats)
-tech-stack (get website tech stack)

GitHub Commands

You can run the following commands to manage your GitHub activity:
-github-info (set up a GitHub token)
-github-delete-token (delete GitHub token)
-github-post-standup (post standup)
-github-update-standup (update standup)
-github-post-comment (post a comment)

GitHub Projects

You can run the following commands to manage your GitHub projects:
-github-projects create-project
-github-projects select-project
-github-projects create-column
-github-projects select-column
