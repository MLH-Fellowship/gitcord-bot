Gitcord Bot

The ultimate Discord Bot that provides you with a number of developer tools and helps you manage your Github repos from Discord.

Integrations

GitCord has a few integrations that allow you to retrieve content from external platforms such as Github, Wappalyzer and Lighthouse.

Utilities

GitCord has many different utilities to help you post quickly on Github. Some examples include posting comments on issues and pull requests. You can also know about the lighthouse stats and technologies used by any site!
